HTML5 TAG DESCRIPTION USE
header element represents a container for introductory content or a set of navigational links. use as a container for introductory content
nav the nav tag defines a set of navigation links use as container for your navigation menu
aside the aside tag defines some content aside from the content it is placed in the aside content should be related to the surrounding content
article the article tag specifies independent,self-contained content use for potential sources for the article element like forum post,blog post,story, and comment
section the section tag defines sections in a document,such as chapters,header,footers,or any other sextions of the document use to divide content
time the time tag defines either a time,or a date in the calendar this element can be used as a way to encode dates and time
footer the footer tag defines a footer for a document or section a footer typically contains the author of the document,copyright,information,links to terms of use contact
hr the hr tag defines a thematic break in an HTML page a hr element is used to separate content in an HTML page
